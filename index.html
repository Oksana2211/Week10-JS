Ответы на вопросы

  1. Сколько живет localStorage?

    Ответ: Данные хранящийся в localStorage не имеют срока давности, но могут быть удалены при очистки истории браузера и с помощью JavaScript.

  2. Как посмотреть localStorage в браузере?

    Ответ: Открыть консоль браузера, перейти на вкладку Application, далее в разделе Storage и нажать на LocalStorage.

  3. Можно ли хранить данные пользователя в sessionStorage?

    Ответ: Можно, но следует помнить, что информация из sessionStorage удаляется после закрытия вкладки/браузера.

  4. Как очистить локальное хранилище в браузере?

    Ответ:

    • localStorage.clear()
    • очистка кэша

  5. Для чего нужны куки JS?

    Ответ: Файлы cookie позволяют хранить информацию о пользователе на веб-страницах и передавать их на сервер.

  7. Как работать с cookie?

    Ответ:

    • запись через document.cookie = 'ключ = значение'
    • для получения значений, записанных в куки, можно просто вывести содержимое с помощью document.cookie
    • удаляем кук с помощью установки параментра expires , который указывает на дату в прошлом

  8. Как добавить и получить значение из веб-хранилища?

    Ответ: Storage.setItem('ключ', 'значение') — добавить
    Storage.getItem('ключ', 'значение') — получить
    Storage.removeItem() — удалить элемент

  9. Придумайте ещё минимум 3 ситуации, помимо предложенных в уроке, для чего может быть нужно сохранять данные пользователя и какие?

    Ответ:
    сохранение прогресса обучения/решения задачи
    сохранения адреса доставки/проживания
    сохранения способа и вида оплаты
    сохранение предпочтений пользователя, для предложения нужных товаров и услуг

  10. Какие преимущества использования JSON?

    Ответ:

    • JSON — это компактный формат. С ним большие объемы данных быстро обмениваются между браузером и веб-сервером.
    • Удобство хранения данных.
    • С JSON-файлами можно работать не только методами JavaScript. Почти у всех языков есть инструменты для чтения и генерации данных JSON.

  11. Какие значения могут быть использованы в JSON?

    Ответ: массивы, объекты, булевые значения, строки, числа, null

  12. Как брать данные из JSON?

    Ответ: Для десериализации или парсинга json-объекта в javascript применяется метод JSON.parse().
